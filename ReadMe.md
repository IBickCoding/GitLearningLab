#Git Learning Lab

##Status
This is currently a work in progress. This repository will be changed as we add new content and when finished will reflect all installation instructions required before starting.

##Recommendation
It is recommended that if you have no Git experience to start in the beginner section and work your way up without skipping material. This isn't absolutely necessary, but skills will build upon previous sections and the sections themselves will be referenced.

##Skill Levels

###Beginner
Intended for those who have never used Git before and are new to version control and source control concepts. This is ideal if you are unfamiliar with command line control as well.

###Intermediate
Intended for learners who have a basic understanding of Git commands and everyday Git workflows.

###Advanced
Intended for college level juniors, seniors, and others who want to strengthen their Git skills, explore more advanced features, and become familiar with common development workflows used in professional environments.

###Professional
Intended for software developers and other industry professionals that are already familiar with Git but are looking to find new ways to strengthen their skills or learn new workflows. This level will focus on learning new collaboration strategies, repository management, and best practices.
